# Voice DNA — template

Copy this to `voice-dna.md` and fill it with how *you* actually write. `voice_check.py`
catches the mechanical AI-tells (em-dash density, slop phrases, punctuation collisions);
this file is for the judgment the linter can't make — whether a generated document
sounds like *you*, or like a brochure.

Run it as a second pass, after `voice_check.py` comes back clean: read the CV / cover
letter against this, list every conflict as **major** or **minor**, fix the majors.
The payoff is a document a hiring manager reads as your own thinking, not vibe-coded.

## Your voice in one line

> _e.g._ "Plain, direct, first-person, honest about limits. Says the true thing, then the useful thing."

Write the sentence that captures how you actually sound. Everything below is detail on it.

## DO

- **Active voice, short sentences.** _Give one real example of a line that sounds like you._
- **[Your signature moves]** — the things you do that most writers don't. _e.g._ name gaps
  before the reader finds them; lead with a metric; open with the counter-intuitive point.
- **Concrete, traceable specifics.** Real numbers only. If it isn't verifiable, cut it.
- **[Your punctuation habits]** — _e.g._ em-dashes are fine in structural lines (`Company — Role`)
  but not as flowing asides in prose. Note yours so the linter's dash-density flag makes sense.
- **[Spelling/locale]** — British vs. American, etc.

## DON'T

- **No corporate slop.** The `voice_check.py` banned list is the floor. If it sounds like every
  other CV, cut it.
- **No [the AI tells you personally hate]** — _e.g._ "not just X but Y", triads-for-rhythm,
  hedging throat-clears ("I believe that", "I am confident that").
- **No inflation, ever.** Titles, numbers, dates stay true. Honesty is part of the voice.

## Registers (match to the document)

- **CV bullets:** _how terse? verb-led? metric-carrying?_
- **Cover letters:** _warm or sharp? how formal? salesy is banned — but what replaces it?_
- **Briefs / long-form:** _structured? blunt?_

## Litmus test

Read it aloud. _Write the one-line test that tells you it's yours_ — e.g. "If it sounds like a
person who'd rather be right than impressive, it's mine. If it sounds like a brochure, rewrite it."
